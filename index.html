Research Adversarial Attacks

Adversarial Attack on Image Compression

An adversarial attack on AI image compression that can drop compressed quality from 35 dB to 13 dB

Problem & Business Value

Image Compression can be found in digital technology and social media apps such as facebook, telegram, etc ... In these platform, due to exponential growth of digital media, an efficient Image Compression is needed. In this project, we design a model to trick AI Compression models, to destroy its reconstruction quality (which can be used in privacy protection from unwanted camera). Alongside this project, a defense mechanism is implemented also to protect users from a perturbed image (that can contains malicious perturbation).

My Role & Contributions

End-to-end ownership: from data collection and model building to simulation and experimentation to automatization. Implemented automated adversarial pipeline from a given image.

Technical Deep Dive

  • Data: Scraped several adversarial research articles from major journals and conferences. Handled smooth gradient flow, vanishing gradient to ensure the backpropagation works very well. Implemented robust adversarial attack model pipeline.
  • Models: Compared FGSM (with drop from 35dB to 10.9dB) against PGD (35dB to 11.7dB) with an mse losstype, while still keeping an average perturbation the image up to 3%
  • Loss Function used: In both models (FGSM and PGD), 3 types of loss function are used, which are Mean Squared Error (MSE), Peak to Noise Signal Ratio (PSNR) and Structural Similarity Index Measure (SSIM). While most of deep learning researcher uses mostly MSE, we dare to go beyond these idea.
  • Evaluation: Achieved a drop from 35dB to 6dB in SSIM loss function implementation with PGD (which is the best result up to now), while the psnr loss function dropped up to 10.2 dB only. .
  • Code Quality: Customizable architecture where users can add more loss functions or other idea. Detailed documentation, mathematical background and type hints throughout.

Implementation & Tools

  • Framework: Implemented using PyTorch with CompressAI library for neural image compression models. Modular architecture supporting multiple attack strategies.
  • Automation: Command-line interface with configurable parameters for batch processing. Automated pipeline for generating adversarial examples across different loss functions.
  • Evaluation: Comprehensive metrics tracking including PSNR degradation, SSIM scores, and perturbation analysis. Automated visualization and result saving capabilities.

Similar Adversarial Attack Research

  • Research output: Accepted for oral presentation at Neuroinformatics 2025 conference. The paper will appear in Volume 1241 of the Studies in Computational Intelligence series.
  • Authors: Arofenitra Rarivonjy, Anton Bibin, Razan Dibo, Aleksandr Kolomeitsev, Anh-Huy Phan, and Ivan Oseledets.
